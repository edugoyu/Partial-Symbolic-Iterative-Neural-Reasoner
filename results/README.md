

# Experimental Results & Visualizations

This directory contains the visual outputs generated by evaluating the **$\psi$-NR pipeline**. The visual data is divided into two distinct categories:

## 1. `error_analysis/`
This folder contains grouped bar charts comparing the frequency of specific rule violations (Row, Column, Color, Adjacency) **before** and **after** the Symbolic-Masker is applied. 

* **Blue Bars (Before Symbolic Pass):** Errors present after the purely neural iterative loop (Neuro-Solver + Mask-Predictor) finishes its iterations.
* **Red Bars (After Symbolic Pass):** The residual errors remaining after the final neuro-symbolic pipeline completes.
* *Note:* The filenames denote which subset of rules were active during that specific run (e.g., `error_graph_col_row_iters4.png` means only the Column and Row rules were actively checked by the Symbolic-Masker and the 4 specifies the iterations $K$).

## 2. `final_pipeline_examples/`
This folder contains end-to-end visual step-by-step breakdowns of individual board solves.

### How to read the pipeline graphs:
* **Iterative Rows (Top rows):** Shows the loop between the Solver making a guess and the Masker identifying errors.
  * **Yellow Squares:** Fixed clues currently trusted by the network.
  * **Magenta 'X':** The Neuro-Solver's current predicted placements.
  * **Masker Audit (Right column):** Green boxes indicate where the Mask-Predictor thinks a queen is missing; Blue boxes indicate where it thinks a queen should be removed.
* **Final Row (Bottom row):** The final guess produced by the neural-only loop.
  * **Symbolic-Masker Red Flags:** Cells highlighted with **Red 'X' boxes** indicate queens that violated the active symbolic rules and were stripped from the board.
  * **Final Solution:** **Lime Stars** represent the final layout after the Neuro-Solver makes one last corrective pass over the filtered board.
